This is an example repost — delete `src/content/blog/example-repost.md` once
you're ready to add a real one.

Add an `original` block to the frontmatter and the post page automatically
shows an "originally posted on {source}" callout linking back to the source,
so you don't have to write that note by hand each time — and the text lives
here permanently even if the original link eventually dies.

The body below is the mirrored content itself (or a summary — your call per
post).
